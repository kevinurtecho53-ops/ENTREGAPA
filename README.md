# Tarea Grupal Semana 2

## Objetivos

1. Implementar un programa que imprima un boleto de viaje interprovincial.
2. Implementar un programa que imprima una boleta de pago, incluyendo descuentos, subtotal y total.
3. Implementar un programa que imprima un ticket de operación de un agente bancario.

## Condiciones

- Los datos deben ser leídos de teclado usando `Scanner`.
- Deben utilizarse todos los tipos de datos primitivos.

## Indicaciones

- **Grupos**: 3 integrantes (máximo).
- **Entregables**:
  - a) Video de 3 minutos (máximo) explicando las soluciones propuestas.
  - b) Archivo ZIP conteniendo el Proyecto Java con las soluciones.
- Cada integrante aparecerá en el video explicando un ejercicio.
- Luego de la explicación de cada solución, se deberá introducir un cambio en el código que genere un resultado incorrecto (explicar los resultados).

## Criterios de Evaluación

Se evaluará la correcta implementación de los programas, la explicación detallada de la lógica utilizada, y la demostración de entender las consecuencias de cambios en el código que lleven a resultados incorrectos.

Buena suerte y ¡feliz codificación!
